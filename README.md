# Проект: Анализ погоды в Telegram

## Описание
Наш бот будет предоставлять информацию о погоде в ближайшие сутки.

## Технологии
- Python 3.10+
- aiogram (Telegram Bot API)
- requests (API Weather)
- pandas & matplotlib (Анализ и визуализация)

## Запуск
1. Склонируйте репозиторий.
2. Создайте файл `.env` и добавьте `BOT_TOKEN` и `WEATHER_API_KEY`.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите: `python main.py`
